# 1. 문제를 익숙한 용어로 재정의한다.

- 주문은 i부터 j까지 할 수 있다.
- 각각의 선물 상자에는 인형이 여러개 들어가있다.
- K명에게 남김없이 나누어 주어야 한다.

# 2. 어떻게 해결할지 계획을 세운다.

- 부분합 알고리즘을 사용한다. 왜냐면 어떤 구간의 합을 여러번 참조해야 하기 때문이다.
- 만족하는 구간은, (psum[j] - psum[i]) mod K == 0
- 한번 주문할 수 있다면, 가능한 주문 방법은 몇가지인지 묻고있다.
  - O(n^2)으로 완전 탐색을 수행한다.
- 여러 번 주문할 수 있다면?
  - 재귀 알고리즘을 사용한다.
